# 🤖 AI Testing Agent — Groq Powered

An intelligent testing agent that automatically generates pytest test cases, detects bugs, and fixes Python code — powered by Groq's LLM API and built with Streamlit.

## 🎯 What It Does

Paste any Python code → the agent:

1. **Generates** comprehensive pytest test cases (edge cases, nulls, negatives, boundaries)
2. **Runs** the tests automatically via pytest
3. **Analyzes** failures to identify root cause bugs
4. **Fixes** the code and presents the corrected version
5. **Summarizes** results in a clean dashboard

## 🧠 Agentic Loop Architecture

```
Input Code
    │
    ▼
Generate Tests (Groq LLM)
    │
    ▼
Run Tests (pytest)
    │
    ├── All Pass → ✅ Clean Code
    │
    └── Failures →
            │
            ▼
        Analyze Bugs (Groq LLM)
            │
            ▼
        Generate Fix (Groq LLM)
            │
            ▼
        Re-Test Fixed Code
            │
            ▼
        📊 Agent Summary
```

## ⚙️ Setup & Installation

### 1. Clone the repository
```bash
git clone https://github.com/YOUR_USERNAME/ai-testing-agent.git
cd ai-testing-agent
```

### 2. Create virtual environment
```bash
python -m venv venv
source venv/bin/activate        # Mac/Linux
venv\Scripts\activate           # Windows
```

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

### 4. Add your Groq API key
Create a `.env` file:
```
GROQ_API_KEY=gsk_your_key_here
```
Get your free API key at [console.groq.com](https://console.groq.com)

### 5. Run the app
```bash
streamlit run app.py
```

## 🧪 Example — What the Agent Finds

**Input (buggy code):**
```python
def divide(a, b):
    return a / b          # Bug: crashes on zero

def is_palindrome(s):
    return s == s[::-1]   # Bug: case sensitive

def find_max(lst):
    max_val = 0           # Bug: fails for all-negative lists
    for x in lst:
        if x > max_val:
            max_val = x
    return max_val
```

**Agent output:**
- ✅ 10 test cases generated
- ❌ 3 bugs detected
- 🔧 Fixed code with proper error handling generated automatically
